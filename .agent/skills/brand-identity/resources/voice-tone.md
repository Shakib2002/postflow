# PostFlow Copywriting: Voice & Tone Guidelines

## Brand Personality
* Professional but approachable
* Direct and efficient
* Empathetic to creators

## Guidelines
* Use Title Case for H1/H2.
* Prefer active voice.
* Keep explanations concise.

## Terminology
| Do Not Use | Use Instead |
| :--- | :--- |
| "Utilize" | "Use" |
| "In order to..." | "To..." |
